# 🎯 SGATE - Performance Management Solution
> Modern Vue3-based Performance Management Dashboard

## ⚡ Overview
기업 성과관리를 위한 모던 프론트엔드 솔루션

## 🛠 Tech Stack
- **Framework**: Vue 3 + Composition API
- **SSR**: Nuxt 3
- **Language**: TypeScript
- **State Management**: Pinia
- **Styling**: SCSS
- **Build Tool**: Vite

## 🏗 Architecture
- 컴포넌트 기반 아키텍처
- 서버사이드 렌더링 최적화
- 모듈화된 상태관리
